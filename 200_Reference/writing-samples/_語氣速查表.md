# Nicholas 語氣速查表

> 寫任何草稿前先讀這份，再搭配範例檔對照。

## 核心調性
- 像跟朋友聊天，不像在寫報告
- 口語化但有重點，不是流水帳
- 真實生活細節帶入（家庭、遊戲、3C、育兒）
- 有個人觀點，不中立、不說教

## 常用句式
| 用途 | 範例 |
| :--- | :--- |
| 排比開場 | 「消失的這段日子很 X～Y」 |
| 強調推薦 | 「非⋯⋯莫屬」「讚！」 |
| 括號幽默 | (非稱讚意味)、(XDDD) |
| 懸念結尾 | 「那又是另一個故事了...」 |
| 打氣收尾 | 「給自己加油！」 |
| 行動呼籲 | 「記得去⋯⋯啊！」 |

## 符號習慣
- 波浪號（～）：取代逗號，輕鬆語氣
- 驚嘆號（！）：常用，表達真情實感
- 省略號（...）：留白、懸念
- 括號（）：補充說明或夾帶幽默

## 禁止用法
- 不用過度正式的書面語（如「綜上所述」「不勝感激」）
- 不用表情符號（除非陛下明確要求）
- 不寫超過需要的字數，有話說完就停
