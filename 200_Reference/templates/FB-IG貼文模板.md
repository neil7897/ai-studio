# FB / IG 貼文模板

> 共用於 Facebook 與 Instagram。
> 使用前先讀 `200_Reference/writing-samples/_語氣速查表.md`。

---

## 模板 A：知識分享 / 專業觀點

```
【Hook 開場——用問句或反常識句吸引人】

（1-2 句點出問題或現象）

我自己的做法是這樣——
- 重點一
- 重點二
- 重點三

（1 句個人心得或感受，帶真實溫度）

你有遇過這種情況嗎？留言聊聊！
```

**使用情境**：分享業務、AI、行銷相關專業心得
**字數參考**：150–250 字

---

## 模板 B：個人生活 / 故事分享

```
【開頭點出主角或場景——人、事、物都行】

（1-2 句交代背景，用真實細節）

印象最深的是——
（核心故事，1-3 句，不要拖）

（一句感受或反思收尾，或留懸念「那又是另一個故事了...」）
```

**使用情境**：家庭、遊戲、生活片段、育兒
**字數參考**：100–180 字

---

## 模板 C：產品 / 服務推廣

```
【開頭用痛點問句或使用情境帶入】

如果你也有這個困擾，我來分享一下我的經驗——

（說明服務或產品解決了什麼，用自己親身角度，不要像廣告）

具體來說：
- 好處一（帶數字或時間更有說服力）
- 好處二
- 好處三

有興趣的話可以私訊我，或直接點連結看看！
```

**使用情境**：介紹福昌紹紀服務、合作案例、行銷內容
**字數參考**：180–280 字

---

## 模板 D：互動 / 問問看官

```
【直接拋出一個問題——越具體越好，別問太大】

（2-3 句說明為什麼這個問題讓你想問）

我自己的答案是 XXX，但我很好奇你們怎麼看～

留言告訴我！（或：投票選你的答案）
```

**使用情境**：引發討論、測試受眾喜好、增加互動率
**字數參考**：80–150 字

---

## 通用注意事項
- 開頭第一句是最重要的，沒抓到眼球後面都白寫
- 每段不超過 3 行，FB 長文也要分段
- 波浪號（～）和驚嘆號（！）保留 Nicholas 語氣
- IG 版本如需 hashtag，附在最後一行，不混入內文
